*安装：
	1.配置数据库，管理密码等基本信息
		1.2 修改 /config.php  

	2.访问安装文件
		2.1 /install.php 进行安装,如果看到的【数字全为0】则安装成功!


*访问地址:

	1.微信上墙
		1.1 地址： /index.php
		1.2 后台： /y_admin/index.php （默认帐号密码均为admin）
	2.微信端接口
		2.1 API接口URL： /wx/wx_init.php
		2.2 说明：第三方融合使用原始xml方式
